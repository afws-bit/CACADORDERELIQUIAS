## Spane
O projeto consiste no desenvolvimento de uma engine educacional em linguagem C voltada para a criação de jogos simples e interativos. A proposta busca facilitar o desenvolvimento de jogos didáticos, como jogos de adivinhação de números de 0 a 100, quizzes e desafios lógicos, auxiliando no aprendizado de programação, lógica computacional e resolução de problemas.

A engine fornece estruturas reutilizáveis para gerenciamento de menus, pontuação, entrada de dados e mecânicas básicas de jogos, permitindo que estudantes e desenvolvedores criem aplicações educativas de forma mais rápida e organizada. O projeto também incentiva o aprendizado prático da linguagem C, promovendo acessibilidade ao desenvolvimento de jogos e fortalecimento das habilidades em programação estruturada.
